# women-safety-web-app
Women Safety Web Application that enables users to send their live location instantly, trigger an SOS alert, and make emergency calls to the police with a single click. Designed to provide quick assistance and improve personal safety during emergency situations using modern web technologies.
# Women Safety Web Application

## Overview

A simple and responsive web application designed to enhance women's safety by providing quick access to emergency features.

## Features

* 🚨 SOS Alert Button
* 📍 Real-Time Location Tracking
* 📲 Share Location via WhatsApp
* 📞 One-Click Police Calling
* 🗺️ Embedded Google Maps Location View

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Geolocation API
* Google Maps Embed
* WhatsApp API

## How to Run

1. Download or clone the repository.
2. Open `women safety.html` in any modern web browser.
3. Allow location permission when prompted.
4. Use the emergency buttons as needed.
